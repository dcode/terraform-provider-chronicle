# chronicle_feed_proofpoint_siem (Resource)

Creates a feed from API source type for Proofpoint SIEM log type.

## Example Usage

```terraform
resource "chronicle_feed_proofpoint_siem" "test" {
  display_name = "ppsiem"
  enabled      = false
  namespace    = "one"
  labels = {
    "env" = "one"
  }
  details {
    authentication {
      user   = "XXXXX"
      secret = "XXXX"
    }
  }
}
```
